История чатов

    Индексирование

    Двухстадийная сборка для большого корпуса. Сначала быстро сохраняем документы в knowledge_store и включаем поиск, затем отдельно достраиваем граф.

    Stage 1 обрабатывает все выбранные файлы и подпапки. Stage 2 по умолчанию строит поисковый граф знаний поверх knowledge_store без LightRAG. LightRAG-граф можно запустить отдельно по выбранной теме, коллекции или всему корпусу.

    Построение графа знаний

    Прежняя логика: документы сразу отправляются в тематический LightRAG runtime. Используйте для малого корпуса или точечной проверки.

    Для больших коллекций используйте «Быстрый запуск», иначе построение LightRAG runtime может быть долгим.

    Операция не запущена —

    Выберите режим индексирования.

    Быстрый запуск: выберите папку, чтобы построить search-ready базу без ввода пути вручную.
